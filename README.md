# zip2pdf

Convert zip file to pdf file

## Environment

- Shell:
    - PowerShell
    - bash
- Node.js: `12.21.0`
    - npm: `7.21.1`
    - yarn: `1.22.11`

***

## Usage

```bash
$ node zip2pdf.js <ZipFile>

# => generated: <ZipFile>.pdf
```

***

## Development

### Setup
```bash
# using npm
$ npm i

# or using yarn
$ yarn

# => installed: node_modules
## * node-zip
## * pdfkit
## * sharp
```
